# `src/content/tasks/` — the task bank (procedural generators + items)

The assessment item bank for the 8 measured signals (spec Дел 4 + Прилог A). All
tasks are **digitally auto-scored** (choice / tap / ordering / tap-the-error),
**procedurally generated** with a difficulty parameter, and **original within the
recognised paradigms** — never copied from WISC / Raven's / KABC / Bebras.

Domains: **Gf** matrices & number/shape series · **Gv** mental rotation & spatial
odd-one-out (SVG) · **Gsm** Corsi span (forward all ages; backward from 8) ·
**Gs** timed symbol search · **EF** Tower of London · **Glr** paired-associate
learning · **CT** sequencing / debug / loop / conditional / maze. Visual tasks are
language-neutral (only instructions/UI localise); CT text items are short and
localised across MK · SR · HR · EN.

Scaffolded in Phase 3.01 — content lands in a later v2 phase.
