# `src/content/report/` — report module library (copy)

The localised content library the [report engine](../../lib/report/) assembles
from (spec Дел 9.2 + Прилог C). Each module is data: a `trigger` condition over
the indices/features, multilingual `text` (mk · sr · hr · en), home `activities`,
an IQ UP! `program_hook`, and a dynamic demo-CTA line tied to the child's growth
area.

Module families: **strengths** (per top index), **speed-accuracy style**, **growth
areas + home activities**, and the **STEM bridge** (links spatial + problem-solving
to the STEM message through narrative, never by changing an index formula). Grows
to dozens of modules so two children rarely get the same report.

**Tone (spec Дел 12):** expert but plain, no jargon, for the parent; growth areas
are "room to grow", never deficits. **No score / IQ number / rank** anywhere.

Scaffolded in Phase 3.01 — content lands in a later v2 phase.
